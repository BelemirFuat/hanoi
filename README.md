# Hanoi Kulesi 

## Özellikler:
- Kullanıcı, oyuna başlamadan önce 3 ile 8 arasında disk sayısı seçebilir.
- Diskler fare ile sürükle-bırak yöntemiyle kuleler arasında taşınır.
- Geçersiz hamleler (örneğin büyük diskin küçük diskin üstüne konması) engellenir ve kullanıcı uyarılır.
- Her hamle sayılır ve üst kısımda hamle sayacı olarak gösterilir.
- Diskler sürüklenirken maviye döner, bırakıldığında eski rengine döner.
- Oyundaki tüm diskler hedef kuleye doğru sırayla taşındığında başarı mesajı gösterilir.
- \"Yeniden Başlat\" butonu ile oyun sıfırlanabilir.
